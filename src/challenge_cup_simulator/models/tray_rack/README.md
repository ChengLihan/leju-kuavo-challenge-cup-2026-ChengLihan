# 料盘架子模型

## 规格
- **长度**: 1.2m
- **宽度**: 0.4m
- **高度**: 1.5m
- **层数**: 3层

## 层高
- **第一层**: 0.6m（距地面）
- **第二层**: 1.0m（距地面）
- **第三层**: 1.5m（距地面，顶层）

## 结构
- 4根金属立柱（直径4cm）
- 3层木质架板（厚度3cm）
- 6根横向支撑杆（增加稳定性）
- 木质底座

## 材质
- 立柱和支撑杆：金属（灰色）
- 架板和底座：木质（棕色）

## 使用方法

在场景配置文件中添加：

```yaml
objects:
  - type: "include"
    name: "tray_rack"
    file: "models/tray_rack/model.xml"
    pos: "3.0 0 0"      # 位置 (x, y, z)
    euler: "0 0 0"      # 旋转角度
```

## 注意事项
- 架子是固定的（不可移动）
- 每层架板可以放置料盘
- 架板尺寸：1.2m × 0.4m，可以放置多个料盘
